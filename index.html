Relación entre Hardware y Virtualización

La virtualización permite que múltiples sistemas operativos funcionen sobre un solo equipo físico, compartiendo recursos como CPU, RAM, Disco y GPU. Esto se logra mediante un software llamado hypervisor, que actúa como intermediario entre el hardware y las máquinas virtuales.

🧠 CPU

El procesador se comparte entre varias máquinas virtuales, asignando núcleos o hilos según demanda.

💾 RAM

Cada VM recibe una porción de memoria, pero el hypervisor puede gestionar el uso dinámicamente.

📀 Disco

Las VMs usan discos virtuales, que son archivos en el mismo disco físico del host.

🎮 GPU

La tarjeta gráfica puede compartirse entre VMs o asignarse a una específica (GPU passthrough).
